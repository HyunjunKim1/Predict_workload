# Predict_workload

Prophet 모델을 사용한 ITEM별 생산량 예측 코드

# Needs

1. python 3.7 이상
2. 엑셀 데이터 저장할 pandas
   - pip install pandas
3. 데이터들 가공 및 계산할 numpy
   - pip install numpy
4. Prophet 예측모델
   - pip install Prophet
5. Data 시각화를 위한 Matplotlib
   - pip install matplotlib

디버깅 전 최신버전 pip 설치
pip install --upgrade pip
